Unit-Тесты

1. Unit-тест для метода логирования (log) в ChatServer
< Создает лог-файл перед тестом.
< Записывает тестовое сообщение с помощью метода log().
< Читает файл и проверяет, записалось ли сообщение.
< Удаляет файл после теста.

2. Unit-Тест для ChatClient
< Создает test_settings.txt с портом 12345.
< Читает его, проверяя, что получено 12345.
< Удаляет файл после теста.
< Логирует сообщение.
< Проверяет, что файл логов создан.
< Читает файл и проверяет, что там есть нужное сообщение.

3. Unit-Тест сетевого соединения ChatIntegrationTest
< Запускает мини-сервер на порту 12345 перед тестом.
< Принимает подключение клиента и ждет сообщение.
< Отправляет ответ клиенту (добавляя Echo: ).
< Клиент подключается, отправляет Hello Server, получает Echo: Hello Server.
< Тест проверяет, что ответ от сервера корректный.
< После теста сервер останавливается.

4. Тестирование многопользовательского чата ChatMultiClientTest
< Запускает сервер в фоновом потоке.
< Создает 3 клиента, подключает их к серверу.
< Первый клиент отправляет сообщение: "Client 0: Hello from Client 0".
< Все клиенты, включая отправителя, получают это сообщение через сервер.
< Проверяет, что все клиенты получили его корректно.
< Завершает соединение и закрывает сервер.
